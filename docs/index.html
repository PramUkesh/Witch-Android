View holders built in

Witch will create view holders for all annotated views. This eliminates the need for creating additional view holders for adapter views. Simply use the RecyclerViewBinderAdapter that requires no implementation besides the actual data binding. Simply provide a binder for each item type in adapter.

{% highlight java linenos %} {% include snippets/6.java %} {% endhighlight %}

Since binder will handle binding of all items of type Post in adapter, it's important to separate binder from item. Binder will be updated with a new item before each bind. This happens under the hood and is handled entirely by Witch.
